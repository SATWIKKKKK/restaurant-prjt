# Restaurant Website

A modern, full-stack restaurant website with React frontend and Node.js backend.

## Features

- Modern dark-themed UI
- Responsive design
- Shopping cart functionality
- Order management system
- RESTful API

## Tech Stack

- React + Tailwind CSS
- Node.js + Express
- JSON file storage

## Installation

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd frontend
npm install
npm start
```

## Usage

- Backend: http://localhost:5000
- Frontend: http://localhost:3000
